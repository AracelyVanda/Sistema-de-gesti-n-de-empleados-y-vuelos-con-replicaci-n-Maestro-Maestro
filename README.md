Descripción:

Sistema desarrollado en C# Windows Forms con conexión a MariaDB, diseñado para la administración de empleados y vuelos de una aerolínea. La base de datos utiliza una configuración de replicación Maestro-Maestro, permitiendo la sincronización de la información entre dos servidores.

Objetivo:

Facilitar el registro y administración de empleados y vuelos mediante una aplicación de escritorio conectada a una base de datos distribuida.

Tecnologías Utilizadas:
* C#
* Windows Forms
* Visual Studio Community
* MariaDB
* Replicación Maestro-Maestro

Funcionalidades:
* Empleados
  * Registrar empleados.
  * Consultar empleados.
  * Modificar información.
  * Eliminar registros.
* Vuelos
  * Registrar vuelos.
  * Consultar vuelos.
  * Actualizar información.
  * Eliminar registros.
* Base de Datos
  * Conexión a MariaDB.
  * Replicación Maestro-Maestro.
  * Sincronización automática de datos.

* Visual Studio Community
* .NET Framework
* MariaDB Server

Ejecución:
* Crear la base de datos utilizando los scripts proporcionados.
* Configurar la conexión a MariaDB.
* Abrir el proyecto en Visual Studio.
* Ejecutar la aplicación con F5.
